# 바이브코딩 발표 자료

## AI와 함께하는 직관적 개발

이 프레젠테이션은 바이브코딩(Vibe Coding)에 대한 발표 자료입니다.

### 주요 내용
- 바이브코딩이란 무엇인가?
- 비개발자의 AI 코딩 경험
- 바이브코딩의 가능성과 한계
- 추천 도구 소개 (Lovable, Google AI Studio, Claude Code)
- MCP 서버 활용법

### 사용 방법
- 방향키(← →) 또는 스페이스바로 슬라이드 이동
- Home: 첫 슬라이드
- End: 마지막 슬라이드

### 기술 스택
- HTML5
- CSS3
- JavaScript (Vanilla)
- 완전 반응형 디자인
